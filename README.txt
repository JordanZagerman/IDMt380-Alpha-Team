How to run the sever:
    1. In the terminal, type "node [Your path to server.js file]"
    2. In the browser, type "http://localhost:8080/[Your path to index.html]"

DB Connection:
    - Path in the code should work to let you connect the database
    - In case it's not working, line 14 in server.js will give the location you're currently operating from
      find the path from that current location to the database file.
    - And replace the path in line 20 in server.js
